# csv2tblpdf

## Prerequisites

node >= 14.7.0
npm >= 6.14.7

## Build Setup

``` bash
$ npm install
```

## Usage

``` bash
$ node csv2tblcsv.js [-noheader] <csv-file>

-noheader: NOT handle the first row of CSV as a header.
```
